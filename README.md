# My CRUD - Spring Boot

A CRUD (Create, Read, Update, Delete) application built with Spring Boot, following a layered architecture using Controller, Service, and Repository patterns.

This project demonstrates backend development best practices, including RESTful API design, data persistence with H2 database, and clean code organization.

---

## 🚀 Features

- Full CRUD operations (Create, Read, Update, Delete)
- RESTful API endpoints
- Layered architecture (Controller, Service, Repository)
- In-memory database using H2
- Clean and maintainable code structure

---

## 🧰 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven (or Gradle)
